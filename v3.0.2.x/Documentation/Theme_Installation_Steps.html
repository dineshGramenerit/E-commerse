Step 1 : Installation and Image Size Settings Back to Top

  • Please, read the whole section, before attempting to install the theme.
  • IT IS RECOMMEND TO ALWAYS BACKUP YOUR SYSTEM BEFORE MAKING ANY UPGRADES OR CHANGES!

  • Copy the directories from “themexxx(update)” (provided in to ZIP file) to your opencart root. Copy those directories at your opencart root respectively,
  • e.g Your copy path will be
  • ”admin” (Overwrite this folder)
  • ”catalog” (Overwrite this folder)
  • ”image” (Overwrite this folder)


Now upload ocmod file, go to Extensions > Installer
Use the Upload file button and select the file nutripe.ocmod.zip from package folder.
Use the Upload file button and select the file productslidergrid-option.ocmod from package folder.
productslidergrid-option.ocmod extension allows us to display product as grid or as slider.
Note:- After installation of productslidergrid-option.ocmod ,if undefine index error appears then please edit the respective module and click on save button.(This may occur when it is installed for the first time).


Go to Extensions > Modifications , and hit the blur Refresh button at the top right corner.


  • If you want to enable languages then you can download it from this site http://www.opencart.com .
  • Download language from given url and paste it to your language directory.
  • Language directory root : "admin/language" and "catalog/language" .

  • Now we have to set our Template from Admin panel
  • Under Admin Panel > Extensions > Extensions,
  • Select Themes from Dropdown as shown below,
  • Now click on Edit button of Your Store.
  • Select nutripe into Theme Directory field,
  • Status Enabled,


    For Image Size setting,
    Change Category Image size to : 893 X 200,
    Change product image thumb size to : 585 X 674,
    Change Product image List size to : 278 X 320,
    Change Additional Product image size to : 585 X 674,
    Change Related Product image size to : 278 X 320,
    And Click on “Save” button at top right corner.


Step 2 : Logo settings Back to Top

For Logo setting , Go to  System > Setting > Edit > Images,
From Store Logo you can upload your logo image.
From Theme Background you can upload your theme background image and set background property fixed or cover.


Step 3 : Language Settings Back to Top

To Enable Languages login to your admin panel and do following settings.

For Example To add Spanish Language :
Go to  System > Localisation > Languages,
Click on “Add New” button to add new Language.
Language Name : Spanish
Code : es-es
Locale : es-es
Status : Enable,
Sort Order : as you like.
Click on “Save” button.

Same way you can do the settings for other Languages.

Step 4 : Banner Settings Back to Top

4.1 : Main Banner


For adding Main banners go to Design > Banners.
(disable HP Products Tab by clicking edit button)
Click on “Add New” button to add new banner.
Give Banner Name : Home Page Slideshow
From status select Enabled,
and click on “Add Banner” button.
Give title of your banner and click on image to upload your banner.
Again click on “Add Banner” button and upload another banners.
Now click on “Save” button.




Now to activate your Main Banners go to Extensions > Extensions. Select Modules from Dropdown.
Now select Slideshow, then click on Edit link
Give Module name: Slideshow-HomePage,
From banner select Main Banner,
Give Width & Height (1903 X 812) (your banner Width and height),
From status select Enabled,
Now click on “Save” button.

4.2 : Left Banner


For adding LeftBanner go to Design > Banners.
Click on “Add New” button to add new banner.
Give Banner Name : LeftBanner
From status select Enabled,
and click on “Add Banner” button.
Give title of your banner and click on image to upload your banner.
Now click on “Save” button.


Now to activate your LeftBanner go to Extensions > Extensions. Select Modules from Dropdown.
Now select Banner, then click on Edit link
Give Module name: Left Banner,
From banner select LeftBanner,
Give Width & Height (278 X 320) (your banner Width and height),
From status select Enabled,
Now click on “Save” button.


4.3 : Carousel Banners


For adding Carousel banners go to Design > Banners.
Click on “Add New” button to add new banner.
Give Banner Name : Manufacturers
From status select Enabled,
and click on “Add Banner” button.
Give title of your banner and click on image to upload your banner.
Now click on “Save” button.


Now to activate your Carousel Banner go to Extensions > Extensions. Select Modules from Dropdown.
Now select Carousel, then click on Edit link
Give Module name: Carousel-HomePage,
From banner select Manufacturers,
Give Width & Height (117 X 94) (your banner Width and height),
From status select Enabled,
Now click on “Save” button.
